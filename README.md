## Run Locally

Clone the project

```bash
  git clone https://github.com/HrushiMakode/portfolio
```

Go to the project directory

```bash
  cd portfolio
```

Install dependencies

```bash
  npm install or yarn install
```

Start the server

```bash
  npm run start or yarn start
```
